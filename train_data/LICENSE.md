## Dataset Attribution & License (FaceForensics++)

This project was trained/validated using **FaceForensics++ (FF++)**, a public deepfake dataset released by its authors.

- **Dataset name**: FaceForensics++ (FF++)
- **Authors / Paper**: Rössler et al., *FaceForensics++: Learning to Detect Manipulated Facial Images*, ICCV 2019.
- **Official dataset page (download & terms)**: https://github.com/ondyari/FaceForensics
- **Paper**: https://arxiv.org/abs/1901.08971
